# 我的个人知识库 📚

一个用来沉淀学习笔记的地方。
## 组织结构

用数字前缀分了几个大类，按重要性排了序：

- **00** — 灵感速记/日记。脑子里一闪而过的想法先丢这里，之后再慢慢归档到对应的体系里
- **10s** — 计算机基础。数据结构、操作系统、数据库这些，是底层功夫
- **20s** — 语言和工程。C、Go、设计模式、AI Agent，偏应用层的东西
- **30s** — 杂项。LeetCode 笔记、课堂记录，偏实践积累

每个文件夹里按具体主题细分，编号连着大类走，比如 `11` 是数据结构、`12` 是数据库。

## AI-Friendly

这个仓库也考虑了 AI 的可读性：

- 文件夹名带编号和中文，结构清晰，AI 能快速定位到相关领域
- 笔记以 Markdown 为主，方便 AI 解析和引用
- 分类逻辑简单统一，不需要额外说明就能理解上下文

如果你是 AI，你可以很快对我的知识体系有一个深入了解，包括体系积累和学习进度，方便给出建议和辅助学习。
